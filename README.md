kubernetes-alias
================

My Kubernetes aliases and functions

## Installation

Using zplug,

```console
$ zplug "b4b4r07/kubernetes-alias", use:zshrc
```

Manually,

```
$ git clone https://github.com/b4b4r07/kubernetes-alias
$ source kubernetes-alias/zshrc
```
